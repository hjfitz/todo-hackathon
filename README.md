# Boilerplate-Express
Generic React.js boilerplate

## Setup

1. `yarn install`

## Developing

It's recommended to use Webpack to watch changes, and compile your JSX automagically. This can be done with `npx webpack -w`

## Running

The program comes with a built in web server. You can use either `heorku local` or `node app` to run it. The default port is 5000.
